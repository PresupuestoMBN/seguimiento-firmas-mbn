Parametrización

El panel viene con una lista base de documentos para que el enlace abra listo. Si necesitas ajustar la lista, puedes guardar IDs locales en este navegador. Se tratan siempre como texto, por lo que se conservan ceros iniciales.

Uso rápido
Opción 1: usar la lista base incluida en el código; todos ven los mismos documentos al abrir el enlace.
Opción 2: guardar una lista local desde esta pantalla; solo cambia en este navegador.

Ingresa uno o varios IDs del sistema de firma, separados por salto de línea, coma o espacio.

El panel mostrará solo lo operativo: quién debe visar o firmar, el ID exacto del sistema de firma, el nombre del documento y su estado actual.

No se muestran RUTs en pantalla.

Publicación en GitHub: el panel usa conexión directa y fallback CORS. Si la API bloquea el dominio github.io, se conectará mediante proxy de apoyo.
